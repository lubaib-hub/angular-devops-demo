Amazon EC2

Elastic Compute Cloud – Scalable Cloud Servers

What is Amazon EC2?

Amazon EC2 (Elastic Compute Cloud) is a core AWS service that provides resizable compute capacity in the cloud.

EC2 allows you to launch virtual servers in minutes and scale them based on demand.

Key Features

  • Virtual machine instances
  • Multiple operating systems (Linux, Windows)
  • Auto Scaling support
  • Secure access via Key Pairs
  • Flexible pricing models

Core Components

  • AMI – Machine template
  • Instance Type – CPU & RAM configuration
  • Security Group – Firewall rules
  • EBS Volume – Persistent storage

EC2 vs Traditional Server

Traditional Server Amazon EC2
Requires hardware purchase Cloud-based virtual server
High upfront cost Pay-as-you-go pricing
Manual scaling Automatic scaling available
