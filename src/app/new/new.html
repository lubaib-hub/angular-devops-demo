Explanation

1️⃣ Trigger

The workflow runs automatically when code is pushed to master or develop branches.

2️⃣ Runner

GitHub provides an Ubuntu virtual machine to execute the pipeline.

3️⃣ Build Process

Node.js is installed, dependencies are installed, and Angular is built in production mode.

4️⃣ Deployment

The browser/ output folder is securely copied to EC2. Then files are moved to /var/www/html and Nginx is restarted.

5️⃣ Security

Sensitive values like server IP and SSH private key are stored securely in GitHub Secrets.
